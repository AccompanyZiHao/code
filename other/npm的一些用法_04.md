## 升级

```sh
npm install -g npm
npm install -g npm@版本号
```

## 切换淘宝源

```sh
npm install -g cnpm --registry=http://registry.npm.taobao.org
```

## 查看全局安装包

```sh
npm list -g --depth=0
```
